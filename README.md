# Proton Mail Tauri

Proton Mail Tauri is a desktop app that wraps the Proton Mail web app using the Tauri library.

## Features
- Uses Tauri for low resource consumption
- Notifications for new emails
- Cross platform (Windows, macOS, Linux)
